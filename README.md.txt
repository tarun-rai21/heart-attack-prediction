# Heart Attack Prediction using Machine Learning

## 📌 Problem Statement
Cardiovascular diseases are one of the leading causes of death worldwide.
This project aims to predict whether a patient is at risk of heart disease using clinical and demographic features.

## 📊 Dataset
The dataset contains patient health records with multiple clinical attributes such as age, blood pressure, cholesterol, ECG results, and exercise-induced indicators.

**Target Variable:**
- HeartDisease (1 → Heart disease, 0 → No heart disease)

## 🛠️ Approach
- Data loading and exploration
- Feature and target separation
- Train-test split
- Data preprocessing using ColumnTransformer
- Model training using Random Forest Classifier
- Model evaluation

## 🤖 Model
Random Forest Classifier is used due to its robustness and ability to handle non-linear relationships in structured data.

## 🚀 How to Run
1. Install dependencies: pip install -r requirements.txt
2. Open the notebook: notebook/heart_attack_prediction.ipynb
